These are my configuration files for Emacs.

initialization. The =[[file:elisp/][elisp]]= directory contains two subdirs; =[[file:elisp/modules/][modules]]=, which
holds my modularized configuration files, and =[[file:elisp/packages/][packages]]=, which just contains
packages I downloaded that weren't available from a package archive.
